# 💼 Job Listings Scraper

A Python bot that automatically scrapes job listings
using Playwright and saves them to a clean CSV file.

## 🛠️ Tools Used
- Python
- Playwright
- CSV

## 📊 Output
Clean CSV file with:
- Job Title
- Company Name
- Location

## ▶️ How to Run
pip install playwright
playwright install
python job_scraper.py
